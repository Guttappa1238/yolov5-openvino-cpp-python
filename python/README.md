## Yolov5-Openvino-Python

### Usage
It was tested on Python 3.9.10.
```shell
$ python -m venv /path/to/env`

Linux/mac
$ source /path/to/env/bin/activate`

Windows
$ \path\to\env\Script\activate`

$ pip install -r requirements.txt
```

Then run the script:
```
python main.py
```
